Asciinema

In this asciicast, we show how a specific shell (called shell and defined in default.nix) can be entered. In that shell, we have access to the cty command-line interface.

In particular, we show that most cty commands are used to alter a state file or to explore it. Although the web interface is the most common way for end users to interact with Curiosity’s state, it can be seen as “just” another command to interact with it.

Note: an asciicast is not a video. You can pause it, just like a video, if it goes too fast and you want to take some time to read what’s on the screen, but you can also select text and copy it.

Note: there are two exit commands displayed at the end; one is to quit the shell entered initially with nix-shell at the beginning of the asciicast, the other is to quit the asciinema recording session.
